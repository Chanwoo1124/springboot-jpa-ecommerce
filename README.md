# 🛍️ Spring Boot & JPA 기반 이커머스 프로젝트

## 🔗 🛠 기술 스택

- **Framework**: Spring Boot (Spring Web)
- **ORM**: Spring Data JPA
- **Template Engine**: Thymeleaf
- **Database**: H2 Database

---

## ❓ 만든 이유

**이론으로 배운 기술을 실제 비즈니스 로직에 녹여내고 싶었습니다.**

스프링 부트와 JPA의 개념을 공부하면서, 단순히 강의를 듣는 것만으로는 부족함을 느꼈습니다.
실제 서비스에서 가장 많이 사용되는 **이커머스(쇼핑몰) 도메인**을 직접 구현해보며 다음과 같은 목표를 달성하고자 이 프로젝트를 시작했습니다.

- **객체 지향 설계**: JPA를 활용해 객체와 관계형 데이터베이스(RDB) 간의 패러다임 불일치 해결
- **MVC 패턴 체화**: Controller, Service, Repository로 이어지는 웹 애플리케이션의 전체 흐름 파악
- **실전 감각**: 단순 CRUD를 넘어 주문, 배송, 회원 등 복잡한 연관관계 매핑 연습

그래서 이 프로젝트는 **안정적인 백엔드 기술 습득과 클린 코드 작성**을 목표로 만들었습니다.

---

## 📌 핵심 컨셉

- **웹 계층 (Web Layer)**: Thymeleaf를 활용한 서버 사이드 렌더링(SSR) 구현
- **도메인 모델 패턴**: 비즈니스 로직을 엔티티 내에서 처리하여 객체 지향적인 설계 추구
- **테이터 접근**: Spring Data JPA를 사용하여 반복적인 SQL 없이 생산성 높은 개발 진행
- **개발 환경**: H2 Database를 사용하여 가볍고 빠른 개발/테스트 환경 구축
