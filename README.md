# 🛍️ Spring Boot & JPA 基盤 ECサイトプロジェクト

## 🔗 🛠 技術スタック (Tech Stack)

- **Framework**: Spring Boot (Spring Web)
- **ORM**: Spring Data JPA
- **Template Engine**: Thymeleaf
- **Database**: H2 Database

---

## ❓ 作成背景 (Motivation)

Spring BootとJPAを学習するために作ったプロジェクトです。
講義を見るだけでなく、実際にコードを書いて理解を深めようと思いました。

- **JPA活用**: JPAを使用して、オブジェクトとリレーショナルデータベース(RDB)を繋げました。
- **MVCパターン**: Controller, Service, Repositoryの基本的な流れを練習しました。
- **機能実装**: 注文や会員管理など、実際のサービスに必要な機能を実装しました。


---

**設計**: ドメインモデルパターンを意識した設計
